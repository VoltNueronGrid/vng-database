10. Model Context Protocol (MCP) Integration

10.1 What is MCP?

MCP (Model Context Protocol) is a standard way to expose database capabilities to AI clients like Claude, Cursor, and VS Code. VoltNueronGrid provides 12 MCP tools with safe read/write boundaries.

10.3 Claude Desktop Configuration

Edit ~/Library/Application Support/Claude/claude_desktop_config.json (macOS) or %APPDATA%\Claude\claude_desktop_config.json (Windows):

{ "mcpServers": { "voltnuerongrid": { "command": "cargo", "args": [ "run", "-p", "voltnuerongrid-mcp", "--bin", "mcp-stdio-server", "--release" ], "env": { "VNG_ADMIN_API_KEY": "your-admin-key-here", "VNG_OPERATOR_ID": "operator-1", "VNG_MCP_RUNTIME_PROXY": "true", "VNG_RUNTIME_BASE_URL": "http://127.0.0.1:8080" } } } }

tools/health

Check database health and metrics:

Request: { "method": "tools/health", "params": { "detailed": true } } Response: { "status": "healthy", "version": "0.1.0", "uptime_ms": 123456, "node_count": 3, "replication_lag_ms": 0, "active_connections": 42, "memory_usage_mb": 512, "query_cache_hit_rate": 0.85 }

tools/schema

Inspect database schema:

Request: { "method": "tools/schema", "params": { "schema_filter": "public", "table_filter": "users" } } Response: { "tables": [ { "name": "users", "columns": [ {"name": "id", "type": "INT", "nullable": false, "primary_key": true}, {"name": "name", "type": "TEXT", "nullable": false} ], "indexes": [ {"name": "idx_users_name", "columns": ["name"]} ] } ] }

tools/query

Execute read-only SQL:

Request: { "method": "tools/query", "params": { "sql_query": "SELECT id, name FROM users LIMIT 10", "timeout_ms": 5000, "max_rows": 1000 } } Response: { "columns": ["id", "name"], "rows": [ ["1", "Alice"], ["2", "Bob"] ], "rowcount": 2, "execution_time_ms": 12 }

tools/ddl_create

Create database objects (requires admin + ddl_admin_key):

Request: { "method": "tools/ddl_create", "params": { "object_type": "table", "object_name": "products", "create_sql": "CREATE TABLE products(id INT PRIMARY KEY, name TEXT NOT NULL)", "ddl_admin_key": "your-ddl-key" } } Response: { "success": true, "object_id": "tbl_products_123", "created_at": "2026-04-28T10:30:00Z" }

10.8 Guardrails

Query safety enforcement on read-only tools:

  • ❌ No mutating keywords: DELETE, INSERT, UPDATE, DROP, ALTER, TRUNCATE
  • ❌ No multi-statement queries
  • ❌ Max query size: 10KB
  • ❌ Max result size: 10MB
  • ❌ Timeout: max 30 seconds
  • ✅ Safe SELECT queries only
  • ✅ CTEs, window functions, joins supported
