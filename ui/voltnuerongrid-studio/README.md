# voltnuerongrid-studio (scaffold)

Frontend/UI scaffold placeholder for VoltNueronGrid Studio.

Planned stack:
- React + TypeScript
- gRPC-web / REST integration with `voltnuerongridd`

## Baseline contract scaffold

- Runtime API client contracts are defined in `src/api/types.ts`.
- Minimal REST client wrapper is in `src/api/client.ts`.
- Contract integrity script: `npm run check:contracts`.
